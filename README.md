# 股票教学案例管理系统 - 完整版

## 功能特点

### 1. 文档自动解析
- 上传 Word、PDF、TXT 文档
- 自动提取股票代码（6位数字）
- 自动识别：
  - 压力位（如"压力位：1800"）
  - 支撑位（如"支撑位：1750"）
  - 风险点（如"风险：跌破支撑"）
  - 机会点（如"机会：机构买入"）
  - 印证类型（筹码/机构DC/密码/趋势/量能/K线等）
- 自动添加到案例库

### 2. 案例记录
- 按时间记录所有讲过的股票
- 记录压力/支撑/风险/机会点位
- 支持手动添加和文档自动提取

### 3. 印证案例库
- PPT案例管理
- 标签化：筹码、机构DC、密码、趋势、量能等
- 记录PPT页码，快速定位
- 支持按标签搜索

### 4. 学员反馈
- 学员获利截图管理
- 方法标签：0家K线、筹码、飘带、机构DC
- 记录获利金额

### 5. 图表展示
- 纳指K线走势图
- 在图表上标记教学案例日期
- 直观展示指数低点与教学内容的关系

## 本地运行

```bash
npm install
npm start
```

访问 http://localhost:3000

## 部署到 Vercel

1. 推送到 GitHub
2. 在 Vercel 导入项目
3. 设置 Node.js 运行时
4. 部署

**注意**：Vercel 不支持文件系统写入，文档上传功能需要改用云存储。
